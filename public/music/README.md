# เพลงสำหรับงานแต่งงาน

กรุณาเพิ่มไฟล์เพลงที่ต้องการเล่นในโฟลเดอร์นี้

## ไฟล์ที่รองรับ:
- `wedding-music.mp3` (แนะนำ)
- `wedding-music.ogg`
- `wedding-music.wav`

## วิธีเพิ่มไฟล์:
1. วางไฟล์เพลงในโฟลเดอร์นี้
2. ตั้งชื่อไฟล์เป็น `wedding-music.mp3` (หรือนามสกุลอื่นที่รองรับ)
3. รีเฟรชหน้าเว็บ

## หมายเหตุ:
- ไฟล์ควรมีขนาดไม่เกิน 5MB เพื่อประสิทธิภาพที่ดี
- รูปแบบ MP3 แนะนำเพราะรองรับทุกเบราว์เซอร์
- เครื่องเล่นจะเล่นเพลงอัตโนมัติเมื่อโหลดหน้า
